HOW TO ADD IMAGES TO YOUR PORTFOLIO
====================================

1. Add your project images to the "images/projects/" folder
   - Name them: project1.jpg, project2.jpg, project3.jpg, project4.jpg
   - Supported formats: .jpg, .jpeg, .png, .gif, .webp

2. Upload to GitHub:
   - In GitHub, navigate to your repository
   - Click on "images" folder, then "projects" folder
   - Click "Add file" > "Upload files"
   - Drag and drop your images
   - Commit the changes

3. Your images will automatically appear on your website!

TIPS:
- Keep image sizes reasonable (under 2MB each)
- Use descriptive names if you add more projects
- Edit index.html to add more project cards if needed
